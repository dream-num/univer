## Introduction

This plugin adds a crosshair highlight to the Univer Sheet.

## Usage

### Installation

```shell
# Using npm
npm i @univerjs/sheets-crosshair-highlight

# Using pnpm
pnpm add @univerjs/sheets-crosshair-highlight
```
