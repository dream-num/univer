## Introduction

`@univerjs/sheets-graphics` is a basic in-cell graphics drawing plugin for Univer Sheet.

## Installation

```shell
# Using npm
npm install @univerjs/sheets-graphics

# Using pnpm
pnpm add @univerjs/sheets-graphics
```
