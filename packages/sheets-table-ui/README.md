# @univerjs/sheets-table-ui

### Installation

```shell
npm i @univerjs/sheets-table-ui
```

### Import

```ts
import { UniverSheetTableUIPlugin } from '@univerjs/sheets-table-ui'

univer.registerPlugin(UniverSheetTableUIPlugin);
```
