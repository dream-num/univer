# @univerjs/sheets-table

### Installation

```shell
npm i @univerjs/sheets-table
```

### Import

```ts
import { UniverSheetsTablePlugin } from '@univerjs/sheets-table';

univer.registerPlugin(UniverSheetsTablePlugin);
```
